#  Bug Tracking System

## Project Description

This project is a web-based Bug Tracking System developed using **ASP.NET Core MVC** and **SQL Server**.
It allows users to report, track and manage software bugs efficiently.

The system supports role-based access:

* Admin – manages users and bugs
* Developer – updates bug status
* Member/User – reports bugs

---

## Features

* User Login & Authentication
* Create New Bug
* View Bug List
* Update Bug Status
* Severity Level (Low, Medium, High, Critical)
* Role-based access control
* Database storage using Entity Framework Core

---

## Technologies Used

* ASP.NET Core MVC
* C#
* Entity Framework Core
* SQL Server
* Razor Views
* Bootstrap

---

## Project Architecture

The project follows **3-Layer Architecture**:

1. Presentation Layer – UI (Views)
2. Business Logic Layer – Controllers
3. Data Layer – Models & Database

---

## How to Run the Project

1. Clone the repository
2. Open in Visual Studio
3. Update database using migrations
4. Run the project

---

## Output

Users can report bugs and track their status in real-time through the web interface.
